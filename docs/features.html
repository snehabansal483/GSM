User Authentication

Secure user authentication system with registration and login functionality.

  • User registration with validation
  • Secure login with session management
  • Password protection
  • Session timeout handling

Inventory Management

Complete control over your grocery store inventory.

  • View all products with details (name, price, quantity)
  • Add new products to inventory
  • Edit existing product information
  • Remove products from system
  • Search and filter products

Order Processing

Efficient order management system for your store.

  • Create new orders with multiple products
  • View all order history
  • Filter orders by date, customer, or status
  • Order summary with totals
  • Automatic inventory updates when orders are placed

Unit of Measure Management

Manage different measurement units for your products.

  • Define custom units of measure (Each,Kg)
  • Associate units with products
  • Convert between units when needed
  • Standardize measurements across your inventory

REST API

Complete CRUD operations via REST endpoints for integration with other systems.

  • Standard RESTful API design
  • JWT authentication for API access
  • Comprehensive documentation
  • Support for all core functionality
